# NeighborNet AI

Community Hero – Hyperlocal Problem Solver

NeighborNet AI is a modern community-driven web application that empowers citizens to report, track, and monitor local civic issues. The platform enables communities to collaborate and improve neighborhoods by providing a transparent system for reporting and resolving public problems.

## 📖 Project Overview

NeighborNet AI helps citizens report local issues such as road damage, garbage accumulation, water leakage, street light failures, and public safety concerns. The application provides real-time issue tracking, community engagement, and an admin dashboard for managing issue resolution.

## ✨ Features

- 🔐 Google Sign-In using Firebase Authentication
- 📋 Report community issues
- 📍 Manual location input
- 🤖 Smart AI-inspired category & urgency suggestions
- 📊 Real-time Dashboard
- 📰 Community Feed
- 🏆 Community Hero Leaderboard
- 👤 User Profile
- 🔎 Search and filter issues
- 🌙 Light, Dark & System Theme
- 📱 Fully Responsive Design
- ⚡ Real-time Firestore integration
- 👑 Admin-controlled issue status management

## 🧠 AI Smart Suggestions

NeighborNet AI includes an intelligent rule-based classifier that automatically suggests:

- Issue Category
- Urgency Level

based on keywords entered in the issue description, providing users with a faster and smarter reporting experience.

## 🎯 Problem Statement

Citizens often lack a simple, transparent, and centralized platform to report local civic issues and track their resolution. NeighborNet AI bridges this gap by enabling collaborative community reporting and transparent issue management.

## 💡 Solution

NeighborNet AI provides an easy-to-use platform where communities can report issues, monitor progress, and encourage faster resolution through transparency and participation.
